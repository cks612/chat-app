### React Socket으로 만든 채팅 App입니다.

## **실행 방법**

### Terminal:

```sh
- git clone

- npm i

- server folder : node index.js

- client folder : npm start
```

## **기술 스택**

```
1. React.js
2. Sass
3. Javascript
4. Socket
5. nodemon
```
